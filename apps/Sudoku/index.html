A classic logic puzzle for daily play

Sudoku challenges you to complete a 9×9 number grid using logic rather than guessing. The planned Android game is designed for quick sessions, steady improvement and comfortable play across different difficulty levels.

Use candidate notes, request hints when needed, follow your progress and return for daily puzzles that help develop concentration, pattern recognition and problem-solving skills.
